## Step Debugger

설문조사 패턴 등에서 페이지 흐름을 쉽게 파악할 수 있도록 커스텀 훅과 디버거를 작성

### 실행
```bash
yarn && yarn dev
```
### 디버거 사용 방법

브라우저에서 개발자 도구(F12)를 실행하면 우측 하단에 벌레 모양의 아이콘이 노출된다. 아이콘을 클릭하면 나오는 플로우 차트를 통해 페이지의 흐름을 쉽게 파악할 수 있다. 또한 다이어그램을 클릭하면 해당 스탭으로 이동할 수 있다.
